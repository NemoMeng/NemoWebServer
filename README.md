# NemoWebServer
一个简单的web服务器<br />
用来加深对网络请求处理的理解<br />
目前NemoWebServer只支持静态资源的处理<br />

1、服务启动方法：<br />
    直接运行HttpServer类<br />
2、测试请求地址：<br />
    http://localhost:8080/index.html<br />
3、静态资源地址：<br />
    webroot/<br />
4、配置相关：<br />
    com.nemo.server.bean.GlobalParams.java<br />
